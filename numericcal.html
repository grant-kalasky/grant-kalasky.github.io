Numericcal UI/UX Redesign

Interaction Designer, September - December 2016

Numericcal is an early-stage startup that enables the design of high-performance digital signal processing for timing-aware applications such as control and communication systems.

Over Fall '16, I worked at Numericcal designing an intuitive, desktop user interface for the company's accelerator customization platform. The interface allows users to analyze/optimize high performance digital signal processing systems. Some of it is internal - I'd be happy to talk more about the project in person if you drop me a line!
